# Packages

- [Yarn Install](https://classic.yarnpkg.com/en/docs/cli/install)
  - _--no-lockfile_ - Don’t read or generate a yarn.lock lockfile.
  - _--pure-lockfile_ - Don’t generate a yarn.lock lockfile.
  - _--frozen-lockfile_ - Don’t generate a yarn.lock lockfile and fail if an update is needed.
